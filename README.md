# ELYU Trabaho: Construction Workforce Management and Recruitment System

## Project Overview
Ang **ELYU Trabaho** ay isang React-based web application na idinisenyo upang tugunan ang mga hamon sa pamamahala ng workforce at recruitment sa sektor ng konstruksyon sa La Union. Nagbibigay ito ng digital na solusyon na nagkokonekta sa mga kumpanya ng konstruksyon sa mga skilled workers, na nagpapadali sa proseso ng paghahanap ng trabaho at pagkuha ng empleyado.

## Key Features
*   **User Authentication**: Role-based access para sa Job Seekers, Employers, at Admins.
*   **CRUD Operations**: Kumpletong Create, Read, Update, at Delete functionality para sa mga trabaho, proyekto, aplikasyon, at user profiles.
*   **Job Posting & Application**: Madaling pag-post ng trabaho para sa employers at pag-apply para sa job seekers.
*   **Project Management**: Pagsubaybay sa mga proyekto at team members.
*   **Data Persistence**: Gumagamit ng `localStorage` para i-save ang data, na mainam para sa demo at prototyping.
*   **Dynamic UI**: Responsive at user-friendly interface na gumagamit ng React state management, conditional rendering, at event handling.
*   **Search & Filter**: Advanced search at filtering options para sa mga trabaho at proyekto.
*   **Data Counters**: Dashboard na nagpapakita ng real-time statistics batay sa user role.

## Tech Stack
*   **Frontend**: React 19
*   **Build Tool**: Vite
*   **Routing**: Wouter
*   **Styling**: Plain CSS
*   **Data Storage**: Local Storage

## Installation & Running
```bash
# 1. Clone the repository
git clone https://github.com/adrianedu27/elyu-trabaho.git
cd elyu-trabaho

# 2. Install dependencies
npm install

# 3. Run the development server
npm run dev
```
