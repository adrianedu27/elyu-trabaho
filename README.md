# ELYU Trabaho
Construction Workforce Management and Recruitment System for La Union.

### What is this?
A simple web app built with React to help construction workers find jobs and companies find skilled labor in the ELYU area.

### Tech used
- React 19
- Vite
- Wouter
- LocalStorage for data

### Setup
1. Clone the repo
2. Run `npm install`
3. Run `npm run dev`
