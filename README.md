# ELYU Trabaho
## Construction Workforce Management and Recruitment System

A React-based web application for connecting construction companies with skilled workers in La Union.

### Features
- User authentication (Job Seeker, Employer, Admin)
- Job posting and browsing
- Project management
- Application tracking
- User profiles
- Admin dashboard

### Tech Stack
- React 19
- Vite
- Wouter (routing)
- Plain CSS
- Local Storage

### Installation & Running

```bash
# Install dependencies
npm install
# or
pnpm install

# Run development server
npm run dev
# or
pnpm dev

# Build for production
npm run build
# or
pnpm build
```
